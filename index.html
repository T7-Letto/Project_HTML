Introdução

Olá, pessoal! Estou compartilhando com vocês o que aprendi nas minhas primeiras aulas de HTML. Confesso que no começo parecia tudo muito complicado, mas com a prática e a ajuda dos meus professores, já consigo criar minhas próprias páginas!

O que aprendi até agora?

  • Estrutura básica: Entendi que todo documento HTML precisa de uma estrutura bem definida, com as tags <html>, <head> e <body>. A tag <head> contém informações sobre a página, como o título, enquanto a tag <body> é onde colocamos todo o conteúdo visível.
  • Cabeçalhos: As tags <h1> até <h6> servem para criar títulos de diferentes níveis de importância. Quanto menor o número, maior a importância do título.
  • Parágrafos: A tag <p> define um parágrafo de texto. É como se fosse um "quebra de linha" mais formal.
  • Formatação de texto: Aprendi a usar as tags <mark>, <small>, <i>, <u>, <strong> para destacar palavras ou frases de diferentes maneiras. Por exemplo, <mark> marca um texto com um fundo colorido, <strong> o texto em negrito e <i> coloca o texto em itálico.
  • Listas: As tags <ol>, <ul> e <li> são usadas para criar listas ordenadas e não ordenadas. As listas são ótimas para organizar informações de forma clara e concisa.
  • Links: A tag <a> é fundamental para criar links para outras páginas ou arquivos. Com ela, posso conectar meu site a outras partes da internet.
  • Outras tags: Pesquisei e descobri algumas tags interessantes como <font> (para definir a fonte do texto), <del> (para riscar um texto) e <abbr> (para criar abreviações).

Meu primeiro projeto

Para colocar em prática tudo o que aprendi, criei uma página bem simples. Nela, usei títulos para organizar as informações, parágrafos para contar um pouco do conteudo aprendido, listas para destacar minhas habilidades e links para minhas redes sociais. Usei também as tags de formatação para deixar o texto mais interessante.

Dificuldades e aprendizados

No início, confesso que me perdi um pouco com tantas tags e atributos. Mas com paciência e bastante prática, comecei a entender a lógica e a desenvolver essa página. A parte mais desafiadora foi entender a diferença entre as tags e como combiná-las para criar layouts mais elaborados.
